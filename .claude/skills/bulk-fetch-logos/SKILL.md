# bulk-fetch-logos

Use this when a dividend module's tickers are missing logo files under
`public/logos/`. This happens when a new module is created or new tickers
are added to an existing universe. Each module has (or should have) a
dedicated `scripts/bulk-fetch-<module>-logos.mjs` that runs the standard
pipeline for its universe.

## When to invoke

- "We're missing logos on [page]"
- "Fetch logos for the quarterly-dy / cashflow / [new module]"
- A new universe file was added and no logo fetch script exists yet
- A new batch of tickers was added to an existing universe

Do NOT invoke for:
- Replacing a single known-bad logo — use `update-logo.mjs <TICKER> <URL>` instead.
- Logos that already exist (the script skips them by default unless `--force`).

## How logos work in this project

- All logos live in `public/logos/<TICKER>.<ext>` (PNG, SVG, JPG, WebP).
- Ticker names are sanitised: `String(ticker).replace(/[^A-Za-z0-9._-]/g, '_').toUpperCase()`.
  Example: `BP.L` → `BP_L.png`, `AP-UN.TO` → `AP-UN_TO.png`.
- A manifest at `src/services/logoManifest.ts` maps safe-ticker → path.
  Regenerated by `scripts/regenerate-logo-manifest.mjs` — the fetch scripts
  call it automatically on success.
- The cron scripts read `public/logos/` at runtime and populate `logoUrl` on
  each pick, so **no Firestore write is needed** after adding logo files.
- After any logo script runs successfully: `git add public/logos/ src/services/logoManifest.ts && git push` to trigger a Netlify rebuild.

## Fetch pipeline (same for every module)

For each missing ticker:
1. **Yahoo profile** (`quoteSummary` → `summaryProfile.website`) — resolves the company's domain and long name.
2. **Gemini suggestion** — batch call (10 tickers per prompt) asking for the best public logo URL (Wikimedia Commons or static CDN preferred). Skipped if no `GEMINI_API_KEY`.
3. **Clearbit** — `https://logo.clearbit.com/<domain>`
4. **Google favicon** — `https://www.google.com/s2/favicons?sz=256&domain=<domain>`
5. **DuckDuckGo favicon** — `https://icons.duckduckgo.com/ip3/<domain>.ico`

First successful download wins. Validation: must be `image/*` content-type, size 1 B – 2 MB.

## Steps

### 1. Check what already exists

```bash
node -e "
const { <UNIVERSE_EXPORT> } = await import('./scripts/<universe-file>.mjs')
const fs = await import('node:fs')
const files = fs.default.readdirSync('./public/logos').map(f => f.slice(0, f.lastIndexOf('.')).toUpperCase())
const fileSet = new Set(files)
const missing = <UNIVERSE_EXPORT>.filter(e => {
  const safe = String(e.ticker ?? e).replace(/[^A-Za-z0-9._-]/g, '_').toUpperCase()
  return !fileSet.has(safe)
})
console.log('Missing:', missing.length)
missing.forEach(e => console.log(' ', e.ticker ?? e))
" --input-type=module 2>&1
```

Adapt `<UNIVERSE_EXPORT>` and `<universe-file>` to the target module:
- AI Picks spine: `{ SPINE }` from `eu-dividend-universe.mjs` (plain string array — use `e` not `e.ticker`)
- Cashflow: `{ CASHFLOW_UNIVERSE }` from `cashflow-universe.mjs`
- Quarterly DY: `{ QUARTERLY_DY_UNIVERSE }` from `quarterly-dy-universe.mjs`

### 2. Use an existing script if one exists

Check if a dedicated fetch script already exists:
```bash
ls scripts/bulk-fetch-*-logos.mjs
```

Existing scripts:
- `bulk-fetch-spine-logos.mjs` — AI Picks SPINE
- `bulk-fetch-quarterly-dy-logos.mjs` — Quarterly DY universe

If a script exists, run it:
```bash
node --env-file=.env.local scripts/bulk-fetch-<module>-logos.mjs 2>&1
```

### 3. If no script exists — write one

Create `scripts/bulk-fetch-<module>-logos.mjs` using this template. Adapt
the three lines marked `# ADAPT`:

```js
// One-time logo fetch for tickers in <universe-file>.mjs that don't yet
// have a file under public/logos/.
import { <UNIVERSE_EXPORT> } from './<universe-file>.mjs'  // # ADAPT
import { GoogleGenAI } from '@google/genai'
import YahooFinance from 'yahoo-finance2'
import { mkdirSync, readdirSync, writeFileSync } from 'node:fs'
import { dirname, resolve } from 'node:path'
import { fileURLToPath } from 'node:url'
import { spawnSync } from 'node:child_process'

const yahooFinance = new YahooFinance({ suppressNotices: ['yahooSurvey'] })
const geminiKey = process.env.GEMINI_API_KEY?.trim() || process.env.VITE_GEMINI_API_KEY?.trim() || ''
const ai = geminiKey ? new GoogleGenAI({ apiKey: geminiKey }) : null

const projectRoot = resolve(dirname(fileURLToPath(import.meta.url)), '..')
const logosDir = resolve(projectRoot, 'public', 'logos')
mkdirSync(logosDir, { recursive: true })

function safeTicker(t) {
  return String(t).replace(/[^A-Za-z0-9._-]/g, '_').toUpperCase()
}
function extFromContentType(ct) {
  if (!ct) return 'png'
  if (ct.includes('svg')) return 'svg'
  if (ct.includes('jpeg') || ct.includes('jpg')) return 'jpg'
  if (ct.includes('webp')) return 'webp'
  return 'png'
}
function stripFences(text) {
  let t = text
  if (t.startsWith('```')) t = t.replace(/^```(?:json)?\n?/, '').replace(/\n?```$/, '').trim()
  return t
}
async function callGeminiWithRetry(prompt) {
  for (let attempt = 0; attempt < 4; attempt++) {
    try {
      const res = await ai.models.generateContent({ model: 'gemini-flash-latest', contents: prompt })
      return (res.text ?? '').trim()
    } catch (err) {
      const isRateLimit = err instanceof Error && err.message.includes('429')
      if (!isRateLimit || attempt === 3) throw err
      const suggestedSeconds = err.message.match(/"retryDelay":"(\d+)s"/)?.[1]
      const delay = suggestedSeconds ? (+suggestedSeconds + 2) * 1000 : 2000 * 2 ** attempt
      console.log(`[gemini] rate-limited, retrying in ${Math.round(delay / 1000)}s…`)
      await new Promise((r) => setTimeout(r, delay))
    }
  }
  return ''
}
function existingLogoFile(safeT) {
  try {
    return readdirSync(logosDir).find((f) => f.toUpperCase().startsWith(`${safeT}.`)) ?? null
  } catch { return null }
}
async function fetchDomainFromYahoo(ticker) {
  try {
    const summary = await yahooFinance.quoteSummary(ticker, { modules: ['summaryProfile', 'price'] })
    const website = summary.summaryProfile?.website
    const name = summary.price?.longName ?? summary.price?.shortName ?? ticker
    if (!website) return { website: null, name }
    const clean = String(website).trim().replace(/^https?:\/\//, '').replace(/\/+$/, '').split('/')[0]
    return { website: clean || null, name }
  } catch { return { website: null, name: ticker } }
}
async function fetchLogoSuggestions(items) {
  if (!ai) return []
  const list = items.map((c) => `- ${c.ticker} | ${c.name} | website: ${c.website || 'unknown'}`).join('\n')
  const prompt = `For each company below, return the single best public URL to its official logo.\nRequirements:\n- Prefer PNG or SVG with transparent background.\n- Prefer Wikimedia Commons or the company's own static asset CDN.\n- The URL must return the image directly (not an HTML page).\n- Return ONLY raw JSON, no markdown:\n[{"ticker":"<ticker>","logoUrl":"https://..."}]\n\nCompanies:\n${list}`
  try {
    const raw = stripFences(await callGeminiWithRetry(prompt))
    const parsed = JSON.parse(raw)
    return Array.isArray(parsed) ? parsed : []
  } catch { return [] }
}
function logoCandidates(geminiUrl, website) {
  const out = []
  if (geminiUrl) out.push(geminiUrl)
  if (website) {
    out.push(`https://logo.clearbit.com/${website}`)
    out.push(`https://www.google.com/s2/favicons?sz=256&domain=${encodeURIComponent(website)}`)
    out.push(`https://icons.duckduckgo.com/ip3/${website}.ico`)
  }
  return Array.from(new Set(out))
}
async function downloadAndValidate(url) {
  const res = await fetch(url, { redirect: 'follow', headers: { 'user-agent': 'Mozilla/5.0 (compatible; winflation-logo-bot/1.0)', accept: 'image/*,*/*;q=0.8' } })
  if (!res.ok) throw new Error(`HTTP ${res.status}`)
  const ct = (res.headers.get('content-type') || 'image/png').split(';')[0].trim()
  if (!ct.startsWith('image/')) throw new Error(`non-image: ${ct}`)
  const buf = Buffer.from(await res.arrayBuffer())
  if (buf.length === 0 || buf.length > 2 * 1024 * 1024) throw new Error(`invalid size: ${buf.length}`)
  return { buf, ct }
}

// # ADAPT: extract tickers from the universe (array of objects or plain strings)
const allTickers = <UNIVERSE_EXPORT>.map((e) => e.ticker ?? e)  // # ADAPT
console.log(`[logos] Universe: ${allTickers.length} tickers.`)

const needFetch = allTickers.filter((t) => !existingLogoFile(safeTicker(t)))
console.log(`[logos] Missing: ${needFetch.length}. Fetching now…\n`)
if (needFetch.length === 0) { console.log('[logos] All logos present.'); process.exit(0) }

console.log('[logos] Resolving domains via Yahoo…')
const domainResolved = await Promise.all(needFetch.map(async (ticker) => ({ ticker, ...(await fetchDomainFromYahoo(ticker)) })))

const geminiSuggestions = new Map()
if (ai) {
  const chunkSize = 10
  for (let i = 0; i < domainResolved.length; i += chunkSize) {
    const chunk = domainResolved.slice(i, i + chunkSize)
    console.log(`[logos] Gemini chunk ${Math.floor(i / chunkSize) + 1}/${Math.ceil(domainResolved.length / chunkSize)}…`)
    const suggestions = await fetchLogoSuggestions(chunk)
    for (const s of suggestions) { if (s.ticker && s.logoUrl) geminiSuggestions.set(s.ticker.toUpperCase(), s.logoUrl) }
  }
} else {
  console.log('[logos] No GEMINI_API_KEY — using Clearbit/favicon only.')
}

const successes = [], failures = []
for (const item of domainResolved) {
  const safe = safeTicker(item.ticker)
  const geminiUrl = geminiSuggestions.get(safe) ?? geminiSuggestions.get(item.ticker)
  const candidates = logoCandidates(geminiUrl, item.website)
  if (candidates.length === 0) { failures.push({ ticker: item.ticker, reason: 'no candidates' }); console.warn(`[logos] ✗ ${item.ticker}: no candidates`); continue }
  let saved = false
  for (const url of candidates) {
    try {
      const { buf, ct } = await downloadAndValidate(url)
      const ext = extFromContentType(ct)
      writeFileSync(resolve(logosDir, `${safe}.${ext}`), buf)
      console.log(`[logos] ✓ ${item.ticker} → ${safe}.${ext} (${buf.length}B)`)
      successes.push(item.ticker); saved = true; break
    } catch { /* try next */ }
  }
  if (!saved) { failures.push({ ticker: item.ticker, reason: `all ${candidates.length} candidates failed` }); console.warn(`[logos] ✗ ${item.ticker}: all candidates failed`) }
}

if (successes.length > 0) {
  console.log('\n[logos] Regenerating manifest…')
  spawnSync('node', [resolve(projectRoot, 'scripts', 'regenerate-logo-manifest.mjs')], { stdio: 'inherit' })
}

console.log(`\n[logos] Done. ✓ ${successes.length}  ✗ ${failures.length}`)
if (failures.length > 0) {
  console.log('\nFailed (fix manually with update-logo.mjs):')
  failures.forEach((f) => console.log(`  - ${f.ticker}: ${f.reason}`))
}
if (successes.length > 0) {
  console.log('\nNext: git add public/logos/ src/services/logoManifest.ts && git push')
}
```

### 4. Run the script

```bash
node --env-file=.env.local scripts/bulk-fetch-<module>-logos.mjs 2>&1
```

This takes 1–3 minutes depending on universe size and Gemini rate limits.

### 5. Commit and push

```bash
git add public/logos/ src/services/logoManifest.ts
git commit -m "logos: backfill <module> universe"
git push
```

This triggers a Netlify rebuild. Logos will appear live within a few minutes.

## Output format for the user

```
✅ Logos fetched: ✓ 46  ✗ 0
   Manifest regenerated: 128 entries
   Next: git add public/logos/ src/services/logoManifest.ts && git push
```

If there are failures:
```
⚠️  Logos fetched: ✓ 40  ✗ 6
   Failed tickers (fix manually with update-logo.mjs):
     - SOME.TICKER: all 3 candidates failed
   ...
```

## Notes / edge cases

- **Gemini daily quota (20 req/day)**: each chunk of 10 = 1 call. A 46-ticker run = 5 Gemini calls. If the AI picks cron already ran today (~11 calls), you may be at or near the limit. Gemini is optional — Clearbit alone covers most major US/EU companies.
- **TSX `.TO` tickers**: Clearbit often fails; Gemini suggestion or the company's own CDN usually works. If all candidates fail, use `update-logo.mjs <TICKER> <URL>` with a Wikimedia URL.
- **`--force` flag**: not implemented in the module scripts by default (unlike `bulk-fetch-spine-logos.mjs`). To re-fetch an existing logo, delete the file first or use `update-logo.mjs`.
- **Safe ticker encoding**: `BP.L` → `BP_L`, `AP-UN.TO` → `AP-UN_TO`. The dot in `BP.L` becomes underscore but periods within tickers like `BRK.B` are preserved — only characters outside `[A-Za-z0-9._-]` are replaced.
